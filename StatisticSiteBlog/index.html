Tentang Proyek Ini

Proyek ini memanfaatkan Azure for Students (Free Tier) untuk meng‐host situs statis. Konten presentasi dibuat dengan Canva, kemudian diupload ke Azure Blob Storage (Static Website).

Ini adalah project tugas akhir mata kuliah Cloud Computing Saya, Az Zahra Lishani Salsabila dengan NIM 22076031. Saya tahu bahwa project ini adalah project sederhana, namun saya berharap ini dapat menjadi contoh bagaimana memanfaatkan layanan cloud untuk menghosting konten pembelajaran secara efisien dan mudah diakses.
